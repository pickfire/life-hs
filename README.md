Haskell Game of Life
====================

Simple Game of Life simulation implemented while learning Haskell.

Usage
-----
Compiled:

    $ ghc -dynamic Main.hs
    $ ./Main

Interpreted:

    $ runghc Main.hs

Interpreted (the long way that I did):

    $ ghci Main.hs
    > main
